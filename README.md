# surface-texture-cnn-matlab
CNN tabanlı yüzey dokusu sınıflandırma (MATLAB)

## Proje Yapısı
- `dataset/wood`: Ahşap doku verileri.
- `dataset/metal`: Metal doku verileri.
- `dataset/fabric`: Kumaş doku verileri.
- `dataset/plastic`: Plastik doku verileri.
- `matlab`: MATLAB kodları ve deney dosyaları.
- `results`: Çıktı görselleri ve sonuçlar.
- `report`: Rapor ve dokümantasyon.
